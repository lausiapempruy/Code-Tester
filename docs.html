Mulai Sekarang

Apa itu VeaCode 2.0?▾
VeaCode 2.0 adalah code playground lokal-first — artinya semua data tersimpan di perangkatmu, tidak ada server backend. Kamu bisa menulis HTML, CSS, JavaScript, JSON, dan Python langsung di browser dan melihat hasilnya secara real-time. Tidak perlu install apapun.
Apakah harus punya akun untuk menggunakan editor?▾
Tidak. Kamu bisa langsung buka home.html dan mulai coding tanpa akun. Akun hanya diperlukan jika kamu ingin menggunakan fitur dashboard, menyimpan preferensi, atau mengakses Code Projects di masa depan.
Bagaimana cara buat akun?▾
Buka halaman daftar dan ikuti 3 langkah:
  1. Isi profil — username (wajib), display name & avatar (opsional)
  2. Buat PIN 10 digit — hafalkan, tidak bisa dipulihkan
  3. Setujui Terms of Service
Sistem butuh 10–20 detik untuk menyiapkan akun ke localStorage.
Apakah VeaCode 2.0 bekerja offline?▾
Ya, editor sepenuhnya berjalan offline setelah halaman dimuat. Font Google Fonts memerlukan koneksi untuk pertama kali, tapi editor itu sendiri tidak perlu internet untuk run preview.

Editor & File

Format file apa saja yang didukung?▾
VeaCode 2.0 mendukung lima format: .html, .css, .js, .json, dan .py. Untuk Python, VeaCode menampilkan konten sebagai preview teks karena tidak ada runtime Python di browser.
Bagaimana cara menambah file baru?▾
Klik tombol + di sebelah kanan tab editor. Isi nama file dan pilih tipe ekstensi, lalu klik Buat. File akan langsung muncul di tab dan sidebar.
Bagaimana cara kerja Run preview?▾
Saat kamu klik Run (atau tekan Ctrl+Enter), VeaCode menggabungkan semua file CSS ke dalam <style> dan semua file JS ke dalam <script>, lalu menyuntikkannya ke dalam HTML utama. Hasilnya dirender di iframe preview di sebelah kanan.

Semua console.log, console.warn, dan console.error dari kode kamu akan tertangkap dan ditampilkan di panel Console di bawah preview.
Apakah kode saya tersimpan otomatis?▾
VeaCode 2.0 adalah testing playground — tidak ada fitur simpan kode permanen saat ini. Kode hanya ada di sesi browser yang sedang aktif. Code Projects (simpan permanen) akan hadir di update berikutnya.
Bagaimana cara hapus file?▾
Hover ke nama file di sidebar kiri, lalu klik ikon ✕ yang muncul di kanan nama file. Minimal harus ada 1 file, jadi file terakhir tidak bisa dihapus.
Bagaimana cara resize panel editor dan preview?▾
Drag pembatas vertikal antara editor dan output untuk mengubah lebar. Drag pembatas horizontal antara preview dan console untuk mengubah tinggi. Cukup klik-tahan lalu seret ke posisi yang diinginkan.

Akun & PIN

Mengapa akun tidak bisa dipakai di beberapa perangkat?▾
VeaCode tidak menggunakan backend server. Akun disimpan sepenuhnya di localStorage browser pada perangkat yang digunakan saat mendaftar. Tidak ada sinkronisasi cloud, sehingga akun hanya bisa diakses dari perangkat dan browser yang sama.
Apa yang dimaksud dengan PIN 10 digit?▾
PIN adalah satu-satunya kunci keamanan akun VeaCode kamu. Panjangnya 10 digit angka. PIN digunakan saat: login, ganti username, ganti PIN itu sendiri, dan verifikasi di Settings. Tidak ada mekanisme reset PIN jika kamu lupa.
Apa yang terjadi jika saya lupa PIN?▾
Karena tidak ada backend, PIN tidak bisa dipulihkan. Jika lupa PIN, satu-satunya opsi adalah menghapus data localStorage secara manual (lewat DevTools browser) dan membuat akun baru. Pastikan menghafalkan atau mencatat PIN di tempat yang aman.
Bagaimana PIN disimpan? Apakah aman?▾
PIN di-hash menggunakan SHA-256 sebelum disimpan di localStorage. PIN asli tidak pernah tersimpan dalam bentuk plaintext. Namun karena ini berjalan di browser lokal, keamanan bergantung juga pada keamanan perangkat yang kamu gunakan.
Bisakah saya mengganti username?▾
Bisa. Buka Settings → tab Account → klik ganti username. Kamu akan diminta mengisi username baru dan PIN saat ini sebagai verifikasi. Perubahan berlaku setelah konfirmasi.

Settings

Apa saja yang bisa diubah di Settings?▾
Settings VeaCode memiliki dua tab utama:
  • Account — Foto profil, display name, ganti username (perlu verifikasi PIN)
  • Security & Data — Ganti PIN, ekspor data ke JSON, impor data dari JSON, hapus akun
Bagaimana cara ganti display name?▾
Buka Settings → tab Account → edit kolom Display Name → klik Simpan. Tidak perlu verifikasi PIN untuk mengubah display name.
Bagaimana cara ganti foto profil?▾
Di Settings → tab Account, klik area avatar untuk memilih gambar baru dari perangkat. Mendukung JPG dan PNG, maksimal 2MB. Gambar disimpan sebagai base64 di localStorage.

Keamanan & Data

Bagaimana cara ganti PIN?▾
Buka Settings → Security & Data → klik tombol Ganti PIN. Prosesnya:
  1. Masukkan PIN baru (10 digit)
  2. Masukkan PIN lama sebagai verifikasi
  3. Ulangi PIN baru untuk konfirmasi
Sistem butuh sekitar 10 detik untuk memperbarui PIN. Setelah selesai, kamu akan diminta login ulang.
Apa yang diekspor saat ekspor data?▾
File ekspor berformat JSON dan berisi semua data akun: username, display name, avatar (base64), tanggal dibuat, aktivitas, serta data Code Projects (jika ada). PIN disimpan dalam bentuk hash. Proses ekspor membutuhkan 5–10 detik.
Bagaimana cara impor data?▾
Buka Settings → Security & Data → klik tombol Impor Data, pilih file JSON yang sebelumnya diekspor dari VeaCode. Proses impor membutuhkan 20–30 detik karena memvalidasi seluruh data. Data yang ada akan ditimpa.
Apakah data saya dikirim ke server manapun?▾
Tidak. VeaCode 2.0 tidak memiliki server backend sama sekali. Semua data — akun, PIN, aktivitas — tersimpan hanya di localStorage browser pada perangkat kamu. Tidak ada telemetri, tracking, atau pengiriman data ke luar.

FAQ

Mengapa error JavaScript tidak muncul di preview?▾
Error JS tetap tertangkap — tapi di panel Console, bukan di preview. Lihat panel Console di bawah preview. Status dot preview akan berubah merah jika ada runtime error.
Bisakah saya menggunakan library eksternal seperti jQuery atau React?▾
Bisa, selama kamu menggunakan CDN link di dalam HTML. Contoh: 
<script src="https://cdn.jsdelivr.net/npm/jquery"></script>
Tambahkan tag <script> di dalam file HTML kamu sebelum run.
Kenapa Python tidak bisa dijalankan?▾
Browser tidak memiliki runtime Python bawaan. VeaCode menampilkan konten file .py sebagai preview teks biasa. Untuk menjalankan Python secara real, pertimbangkan menggunakan layanan seperti Replit atau Google Colab.
Apakah VeaCode 2.0 open source?▾
Ya. VeaCode 2.0 tersedia di GitHub. Kamu bebas melihat, memodifikasi, dan mendistribusikan ulang sesuai lisensi yang berlaku. Lihat README.md untuk detail lebih lanjut.
Browser apa yang direkomendasikan?▾
VeaCode 2.0 bekerja paling baik di Chrome 100+, Firefox 100+, dan Edge 100+. Safari juga didukung. Hindari browser lama yang tidak mendukung backdrop-filter dan CSS custom properties.
Bagaimana cara melaporkan bug?▾
Buka issue di repositori GitHub VeaCode. Sertakan langkah reproduksi, browser yang digunakan, dan screenshot jika memungkinkan.
